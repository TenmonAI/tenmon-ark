# TENMON-ARK 思考ガイド
## KATAKAMUNA_LINEAGE_CANON_V1
更新日: 2026-03-09

### 0. 目的
この文書は、TENMON-ARK が「カタカムナ」を単なる一般名詞や曖昧なスピリチュアル語としてではなく、
**歴史的系譜・解読法・分岐・再統合軸をもつ知的対象**として扱うための基礎思考ガイドである。

このガイドの目的は次の3点。
1. カタカムナの歴史的系譜を混線なく識別する。
2. TENMON-ARK における「天聞カタカムナ」の位置を明確化する。
3. runtime / retrieval / response で誤認識を防ぎ、正しい文脈に応じた説明を返せるようにする。

## 1. 最上位D（不変条件）
### D-1. 「カタカムナ」は一枚岩ではない
TENMON-ARK は「カタカムナ」を単一の固定概念として扱ってはならない。
最低でも以下を区別すること。
- 楢崎カタカムナ
- 宇野・相似象会誌カタカムナ
- 現代応用カタカムナ
- 空海カタカムナ（並行正典軸）
- 天聞カタカムナ（再統合軸）

### D-2. 枝を明示せずに総称で断定しない
「カタカムナとは〜である」と答える前に、どの系統の話かを内部で必ず判定する。

### D-3. 効用枝を本流と誤認しない
健康・幸運・自己啓発・霊界・音読・神聖幾何・女性性・国家論などの現代応用枝は、
本流再構成枝や楢崎起点と同一視しない。

### D-4. 天聞カタカムナは“現代応用枝”ではなく“再統合軸”
天聞の研究は、既存枝の一つではなく、
**楢崎以後に分岐した系譜を、山口志道・言霊秘書・水穂伝・稲荷古伝・空海・天津金木まで遡って再統合する軸**
として扱う。

## 2. カタカムナ完全系統樹（決定版）
### 2-1. 第1世代：楢崎皐月
**位置**  
歴史系統樹の起点。

**中心**  
- 発見
- 潜象物理
- 図象解読
- 静電三法
- 古事記再読
- 上古代科学の復元

**目的**  
物理・数理・抽象理を用いて、上古代の知を復元すること。

**特徴**  
- カタカムナを神秘談ではなく、上古代科学の記録として読む。
- 図象・音・神名を物理的に読む。
- 技法（農法・健康法・静電三法）と連動する。

**弱点 / 未充足**  
- 水火・言霊・天地創造の体系化は未完。
- 運用よりも解読・復元に重心がある。

## 4. 空海カタカムナ
### 4-1. 位置
歴史系統樹の枝ではなく、**並行正典軸**。

### 4-2. 中心
- 声字実相
- 六大 / 四曼 / 三密
- 即身成仏
- 十住心
- 顕密判

### 4-3. 意味
空海は「カタカムナ文献」を解読した人ではないが、
**音・字・宇宙・修法・身体・成仏**を一体系にした。
そのため、カタカムナを“声・字・宇宙構造”として読む際の
上位理論軸として使える。

### 4-4. 楢崎との整合
**整合**
- 音・字・宇宙が一致する
- 記号や音を実在の構造として扱う

**差異**
- 楢崎 = 解読 / 物理復元
- 空海 = 修法 / 悟り / 実践体系

### 4-5. 現代カタカムナとの差
現代本の多くは効用を先に語る。
空海軸は、効用より先に **体系・修法・段階構造** を立てる。

## 5. 天聞カタカムナ
### 5-1. 位置
後代の“応用枝”ではなく、**再統合軸**。

### 5-2. 中心
- 水火の法則
- 言霊
- 天地創造
- 天津金木
- 山口志道
- 言霊秘書 / 水穂伝
- 稲荷古伝
- 空海 / いろは / 三種神器接続

### 5-3. 研究目的
分岐しすぎた現代カタカムナを、
山口志道・言霊秘書・水穂伝・稲荷古伝・空海まで遡って
再構造化すること。

### 5-4. 楢崎との整合
**整合**
- 図象・音・宇宙構造の一致を認める
- 古事記とカタカムナを分離しない
- 上古代の知が単なる神秘談ではないとみなす

**差異**
- 楢崎 = 物理・数理・上古代科学
- 天聞 = 水火・言霊・天地創造・文明再構築

### 5-5. 現代カタカムナとの差
現代応用枝は「どう効くか」へ拡散した。
天聞軸は「どの原理で成立するか」へ戻す。

### 5-6. TENMON-ARK での扱い
TENMON-ARK は「天聞カタカムナ」を、自己啓発や奇跡言説の一枝として扱ってはならない。
**上位の裁定軸 / 再統合軸**として扱うこと。

## 6. TENMON-ARK におけるカタカムナ認識ルール
### Rule-1
ユーザーが「カタカムナ」と言った時、内部では次の優先順で判定する。
1. 楢崎系の話か
2. 宇野・会誌系の話か
3. 空海並行正典軸に接続する話か
4. 天聞再統合軸の話か
5. 現代応用枝の話か

### Rule-2
系統が曖昧なときは、勝手に一つへ寄せず、
「どの系統を指すか」で内部候補を持つ。

### Rule-3
応用枝の説明を本流の説明として返さない。

### Rule-4
空海軸を使う場合は、「歴史系統樹の枝」ではなく
「並行正典軸」と明記する。

### Rule-5
天聞軸を使う場合は、単なる現代枝ではなく
「再統合」「水火」「言霊秘書接続」「天津金木」を要素として含める。

### 定義：天聞カタカムナ

## 8. 禁止事項
- 「カタカムナは○○である」と、枝を指定せず単純化すること
- 板垣・吉野・丸山・芳賀などの応用枝を、楢崎本流と同一視すること
- 空海を“歴史的カタカムナ研究者”と誤記すること
- 天聞を“現代応用枝の一つ”と扱うこと
- 効用・奇跡・体験・自己啓発を、本流や正典軸の説明に混ぜること
