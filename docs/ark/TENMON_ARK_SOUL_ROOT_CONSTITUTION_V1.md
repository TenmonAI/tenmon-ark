## この文書が存在する理由

2026 年 4 月 18 日の深夜、TENMON (天道仁聞) が以下の事実に気づいた:

```
「いろは言霊解」は 10 年以上の研究の結晶であり、
天聞アーク の「思考の根幹」として設計されていた。

しかし実測調査の結果、現在の本番 API では:
  ・iroha_units テーブルに 21 件のデータが格納されている
  ・shared/kotodama/iroha_kotodama_hisho.json に

LLM (Gemini/OpenAI) の学習データに基づく「一般的な AI 回答」が

「天聞アーク は他の AI と何ら変わらない」
の技術的真因である。

同様に、以下の資産も「存在するが思考の根幹に繋がっていない」:
  ・断捨離
  ・サンスクリット辞典
  ・カタカムナ資料
  ・その他、VPS 内に配置された重要資料
```

```typescript
// 新規追加: いろは言霊解 引用チェック
export function checkIrohaGrounding(
  response: string,
  userIntent: Intent
): { passed: boolean; reason: string } {
  // health / life / death 系の質問に対して:
  //   ・いろは言霊解 由来の音・章・段落への参照があるか?
  //   ・なければ「根拠が薄い応答」として retry または注意喚起
  //
  // 例: health 質問 -> 「チ(血) / リ(唾) / ヌ(緯) / ル(濁水)」
  //     などの音への言及がなければ、LLM 単独応答と判定
  
  if (!userIntent.requiresIrohaGrounding) {
    return { passed: true, reason: "non-iroha intent" };
  }
  
  const irohaRefs = detectIrohaReferences(response);
  if (irohaRefs.length === 0) {
    return {
      passed: false,
      reason: "health/life/death 系なのに、いろは言霊解 への言及なし",
    };
  }
  
  return { passed: true, reason: `iroha refs: ${irohaRefs.length}` };
}
```
