## 運用フェーズで見るべき指標（3つ）

### 1. 初回アクセス → 操作開始率（First 10 Seconds）

**指標名**: `first_action_completion_rate`

**測定方法**:
- 初回アクセスから10秒以内に操作（チャット送信、新規チャット作成）を開始したユーザー数 / 初回アクセス総数

**目標値**: 70%以上

**観測ポイント**:
- 10秒以内に操作開始できないユーザーの割合
- 空状態の文言が適切か
- 入力欄の視認性

---

### 2. プラン別機能利用率（Plan Gate Effectiveness）

**指標名**: `plan_gate_interaction_rate`

**測定方法**:
- PlanGate が表示された回数
- Upgrade CTA のクリック率
- プランアップグレード率

**目標値**:
- PlanGate 表示回数: 適切（過多でない）
- Upgrade CTA クリック率: 5%以上
- プランアップグレード率: 2%以上

**観測ポイント**:
- Free/Basic ユーザーが Founder 機能を見た時の反応
- グレーアウト表示が適切か
- Upgrade CTA が押し付けでないか

---

### 3. オフライン使用率（Offline Mode Adoption）

**指標名**: `offline_mode_usage_rate`

**測定方法**:
- OfflineStatusBar が表示された回数
- オフライン時の操作継続率
- オフライン時のエラー発生率

**目標値**:
- オフライン時の操作継続率: 80%以上
- オフライン時のエラー発生率: 5%以下

**観測ポイント**:
- 「個体モードで稼働中」の表示が適切か
- オフライン時も操作が継続できるか
- ユーザーが不安を感じていないか
